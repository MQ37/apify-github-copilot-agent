# Apify Github copilot agent

Apify Github copilot agent integrates Apify's web scraping and automation capabilities easily into your existing projects.

## Instagram CLI Example

This repository includes a practical example of integrating Apify into a Node.js CLI application. The Instagram CLI tool demonstrates how to:

- Use the Apify client to call Instagram scraper Actor
- Process and display scraped data
- Build a user-friendly command-line interface

See [INSTAGRAM_CLI_README.md](INSTAGRAM_CLI_README.md) for detailed documentation on the Instagram CLI tool.

## Quick Start

```bash
npm install
cp .env.example .env
# Add your APIFY_TOKEN to .env
npm start -- -p therock
```
