# Apify Github Copilot Agent

Apify Github Copilot Agent integrates Apify's web scraping and automation capabilities easily into your existing projects.

## Example: Instagram CLI Tool

This repository demonstrates integration of the [Apify Instagram Scraper](https://apify.com/apify/instagram-scraper) into a Node.js CLI application.

### Quick Start

```bash
# Install dependencies
npm install

# Set up your Apify token
cp .env.example .env
# Edit .env and add your APIFY_TOKEN

# Run the CLI
npm start -- -p therock
```

See [INSTAGRAM_CLI_README.md](./INSTAGRAM_CLI_README.md) for detailed documentation.
